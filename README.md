# مركز فينكس للتدريب والاستشارات

واجهة مالية وتشغيلية عربية لمركز فينكس، مبنية خصيصًا للمرحلة الأولى من النظام وفق الوثائق المرجعية.

## ما الذي يتضمنه المشروع؟

- صفحة دخول عربية آمنة خالية من أي بيانات مالية قبل تسجيل الدخول.
- لوحة تحكم احترافية بخط **المراعي** وحجم خط أصغر وأكثر راحة.
- جميع الشاشات الأساسية المطلوبة:
  - نظرة عامة
  - الدورات
  - موظفو المبيعات
  - التسجيلات
  - التحصيلات
  - رسوم التحصيل
  - الاستردادات
  - تحويلات التسجيل
  - التحويلات إلى الخزنة والتحويلات الداخلية
  - المصروفات
  - وسائل التحصيل والحسابات
  - الأطراف والأشخاص
  - التقارير
  - الإعدادات
- دعم مصادقة لمستخدمين أساسيين: المدير والمحاسب.
- رفع مرفقات في الشاشات التشغيلية.
- طباعة التقارير على مقاس أ٤.
- تصدير شاشة التقارير إلى ملف إكسل.
- مخطط قاعدة بيانات جاهز للاستهداف على PostgreSQL.

## التقنيات المستخدمة

- نكست
- تايب سكربت
- تيلويند
- موجّه التطبيقات

## التشغيل المحلي

```bash
npm install
npm run dev
```

## ملاحظات مهمة

- الواجهات الحالية تعمل ببيانات عرض تشغيلية منظمة لإظهار منطق النظام بالكامل.
- تم تجهيز مخطط قاعدة بيانات وبنية مصادقة ورفع مرفقات استعدادًا للربط الخلفي الكامل.
- الملفان التاليان مهمان للتجهيز الخلفي:
  - `prisma/schema.prisma`
  - `docs/database-structure.md`
