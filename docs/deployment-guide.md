# دليل تشغيل ونشر النظام

## المتطلبات

- خادم يدعم تشغيل Node.js
- قاعدة بيانات PostgreSQL
- متغيرات البيئة الموجودة في الملف `.env.example`

## الخطوات المقترحة

1. تثبيت الحزم:
   `npm install`
2. إنشاء ملف البيئة اعتمادًا على `.env.example`
3. إنشاء قاعدة البيانات وتطبيق الترحيل
4. تشغيل المشروع محليًا:
   `npm run dev`
5. بناء نسخة النشر:
   `npm run build`
6. تشغيل نسخة الإنتاج:
   `npm run start`

## ملاحظات مهمة

- صفحة الدخول لا تعرض أي أرقام أو أرصدة قبل التحقق من هوية المستخدم.
- المرفقات تُرفع إلى مجلد عام داخل المشروع في هذه النسخة.
- التقارير مهيأة للطباعة على مقاس أ٤ والتصدير إلى ملف إكسل.
