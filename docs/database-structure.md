# هيكل قاعدة البيانات المقترح

تم تجهيز مخطط قاعدة بيانات يستهدف PostgreSQL ليدعم الإطلاق الفعلي للنظام داخل مركز فينكس فقط.

## الجداول الأساسية

- المستخدمون والصلاحيات
- الأطراف والأشخاص
- وسائل التحصيل والحسابات
- الدورات
- موظفو المبيعات
- التسجيلات
- التحصيلات
- رسوم التحصيل
- الاستردادات
- تحويلات التسجيل
- التحويلات الداخلية والخزنة
- المصروفات
- المرفقات
- سجل المراجعة

## مبادئ التصميم

- حفظ العملة الأصلية وسعر الصرف والمعادل بالجنيه المصري في كل حركة مالية.
- منع الحذف الفعلي والاعتماد على الحالات كلما أمكن.
- دعم المرفقات على مستوى التسجيلات والتحصيلات والاستردادات والتحويلات والمصروفات.
- فصل العمولة المتوقعة عن العمولة المستحقة والمصروفة والملغاة.
- دعم تعدد المستخدمين بصلاحيات: مدير ومحاسب.

## الملفات المرتبطة

- `prisma/schema.prisma`
- `lib/prisma.ts`
- `.env.example`
